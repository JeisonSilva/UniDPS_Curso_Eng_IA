# UniDPS_Curso_Eng_IA

Repositorio com projetos praticos desenvolvidos ao longo do curso de Engenharia de IA para Devs.
O foco deste material e consolidar aprendizado sobre o uso de ferramentas de inteligencia artificial aplicadas ao desenvolvimento de software, com exemplos pequenos, objetivos e evolutivos.

## Visao geral

Este repositorio funciona como uma colecao de estudos orientados a pratica.
Cada projeto explora uma ideia especifica, como treinamento de modelos, preparacao de dados, inferencia, automacao de fluxos ou uso de bibliotecas voltadas a IA.

## Objetivo

Os projetos foram organizados para:

1. Estudar conceitos de IA de forma pratica.
2. Experimentar bibliotecas, modelos e fluxos de trabalho.
3. Entender como usar ferramentas de IA no desenvolvimento de software.
4. Manter exemplos separados por projeto para facilitar consulta e evolucao.

## Como este repositorio esta organizado

Cada pasta representa um projeto independente de estudo.
Quando disponivel, cada projeto possui um `README.md` proprio com contexto, objetivo, estrutura e breve explicacao tecnica.

## Projetos

| Projeto | README | Descricao |
|---|---|---|
| Projeto 1 | [projeto1/README.md](./projeto1/README.md) | Exemplo introdutorio de classificacao com TensorFlow.js em Node.js, usando dados ficticios para treinar um modelo capaz de prever categorias como `premium`, `medium` e `basic`. |
| Projeto 2 | [projeto2/README.md](./projeto2/README.md) | Sistema de recomendacao de produtos com TensorFlow.js, treinando uma rede neural binaria para prever probabilidade de compra e gerar recomendacoes personalizadas por usuario. |
| Projeto 3 | [projeto3/README.md](./projeto3/README.md) | Classificacao binaria de aprovacao de credito com rede neural treinada a partir de regras de negocio. Inclui interface web para consulta ao modelo sem necessidade de backend. |

## Tecnologias exploradas

- Node.js
- TensorFlow.js
- Scripts para preparacao e transformacao de dados
- Fluxos simples de treinamento e previsao

## Evolucao do repositorio

Novos projetos podem ser adicionados a esta estrutura conforme o avancar do curso, mantendo a proposta de documentar de forma clara o que foi estudado, implementado e aprendido em cada etapa.
